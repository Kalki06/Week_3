# Expense Tracker CLI

## Overview
Simple command line tool to track expenses.

## Features
- Add expense
- Delete expense
- Search by category
- Expense summary

## Installation
Python 3.x required

Run:

python expense_tracker.py

## Example Output
1. Add Expense
2. Delete Expense
3. Search by Category
4. Show All Expenses
5. Total Expense
6. Exit

## Design Decisions
Used list of dictionaries to store structured records.

## Limitations
Data not persisted after program exit.
